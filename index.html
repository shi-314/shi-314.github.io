Versus

Versus is an Android application of Versus.com I have worked on. The app is a native Android client and provides a mobile user interface for the optimal user experience.

I have contributed as well to the design as to the implementation of the project. At the moment it has about 500.000 downloads on the Google Play Store.

AngryKings

AngryKings is an online multiplayer 2D game for Android I have created with other students at the university. The goal of the game is to bring down the oponents castle. It uses the Box2D physics engine for animation and the AndEngine for the rendering.

On the server side we used Erlang. The most interesting part of the project is the synchronization of the the physics engine over two devices with limited bandwidth.

ComputerVision

ComputerVision is a small C++ library I created in my spare time with basic image processing algorithms for line detection based on Hough transform. I started the project out of curiosity and for educational reasons. It helped me to understand the basic image processing algorithms and C++ basics.
